SkyMerchant unit tests

This contains the tests for the SkyMerchant project.

  • Stable: Contains all the stable tests and features in the wizards. This should serve as a documentation source for the features you can use in this project
    Tests that have 'DOCUMENTATION' in their comment are tests that should be considered documenting the features.
  • Development: Tests being worked on, not yet useable as documentation or maybe not operational
  • Unused: working features, but not used or required anymore for the project
  • Broken: simply partially implemented code, or code broken somewhere along the line. Might not be usefull, or might need refactoring.
