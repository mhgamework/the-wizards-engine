IDE

Contains ideas on how to extend the visual studio IDE, or programming in general.

  • Find a way to add a context menu item to the solution explorer.

  • Add open or create test class feature. Add option to configure where tests should be located for specific namespaces or folders
  • Add idea manager: Add a new VS window, and in this window you can quickly type an idea, bind it to a tag (provided with intellisense), and access the latest tags using the D1-9 keys.
  • Add a diagram generater that shows code dependencies for a list of types, with a fixed depth (eg show all dependencies 3 deep). Preferably use VS tool window for this.
  • Add solution explorer context item: close all (on folders or selection). Also add open all!
